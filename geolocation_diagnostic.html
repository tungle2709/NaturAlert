  1. System Preferences: Go to System Preferences > Security & Privacy > Privacy > Location Services
  2. Enable Location Services: Make sure "Enable Location Services" is checked
  3. Browser Permissions: Find your browser (Safari, Chrome, Firefox) and enable location access
  4. Website Permissions: In browser settings, allow location access for localhost
  5. WiFi Connection: Connect to WiFi for better location accuracy
  6. Restart Browser: Close and reopen your browser after changing settings

Alternative Solutions

If geolocation still doesn't work, you can:

  • Use the search box to find your city (e.g., "Toronto", "New York")
  • Enter coordinates manually in the format: latitude,longitude
  • Use a popular city as a test location
